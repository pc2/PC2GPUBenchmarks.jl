Example: Host-to-Device Memcpy

Note

Test system: DGX with 8x A100 GPUs

julia> host2device_memcpy()
Memsize: 512.000 MiB
GPU: CuDevice(0) - NVIDIA A100-SXM4-40GB

Host <-> Device Bandwidth (GiB/s):
 └ max: 11.75

Host (pinned) <-> Device Bandwidth (GiB/s):
 └ max: 24.24

Device <-> Device (same device) Bandwidth (GiB/s):
 └ max: 1226.84

Or with a bit more statistics on the benchmark runs:

julia> host2device_memcpy(; stats=true)

GPU: CuDevice(0) - NVIDIA A100-SXM4-40GB

Host <-> Device Bandwidth (GiB/s):
 ├ max: 11.71
 ├ min: 8.74
 ├ avg: 10.2
 └ std_dev: 1.22

Host (pinned) <-> Device Bandwidth (GiB/s):

 ├ min: 21.86
 ├ avg: 23.54

Device <-> Device (same device) Bandwidth (GiB/s):
