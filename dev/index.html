Installation

The package is not registered in the General registry, but only in the PC2Registry. If you have subscribed to the latter, you can simply add the package as follows.

] add PC2GPUBenchmarks.jl

Otherwise, you can readily add it by using the explicit URL.

] add https://git.uni-paderborn.de/pc2/julia/PC2GPUBenchmarks.jl

Note: The minimal required Julia version is 1.6 but we recommend using Julia ≥ 1.7.
